    LES INTERFACES D'ENTRÉE VIDÉO UTILISÉES DANS LES ÉCRANS LCD:

    DVI

    Le DVI est un type de connecteur qui sert à relier une carte graphique à un dispositif d'affichage tel qu'un écran d'ordinateur

    Plus d'info
    High-Definition Multimedia Interface

    Le HDMI permet de relier une source audio/vidéo DRM par câble cuivre ou fibre – comme un lecteur Blu-ray, un ordinateur ou une console de jeu – à un dispositif compatible – tel un téléviseur HD

    Plus d'info
    DisplayPort

    Le DisplayPort est comme le hdmi sauf que lui, il peut transporter jusqu'a 32Gbit/s donc il est utiliser soi pour le gaming pour pouvoir jouer a plus de 100 image par seconde ou alors il est utiliser dans les ecran haute definition.

    Plus d'info
    Mini displayport

    Le Mini DisplayPort (MiniDP ou mDP) est une version miniaturisée de l'interface numérique audiovisuelle DisplayPort. Les fonctionnalités et signaux sont les mêmes.
